# 3Axis_gimbal
基于storm32bgc开发的三轴云台，能够实现云台自稳、用遥控器遥控控制X、Y的转动。

## gimbal文件
基于Qt开发的上位机，基于开源控件Qwt开发的；
功能：
①三轴波形显示；
②指针、油门等功能。

## 云台控制程序
基于Storm32gbc开发的工程文件

主要包括：
①姿态解算；
②pid，spwm控制；
